Tauri'yi bu kadar esnek yapan şeylerden biri, neredeyse tüm ön uç çerçeveleriyle çalışabilmesidir. Resmi olarak korunan çerçeve şablonlarından birini kullanarak yeni bir Tauri projesi oluşturmanıza yardımcı olmak için [`create-tauri-app`](https://github.com/tauri-apps/create-tauri-app) yardımcı programını oluşturduk.

`create-tauri-app` şu anda vanilla (çerçevesiz HTML, CSS ve JavaScript), [Vue.js](https://vuejs.org), [Svelte](https://svelte.dev), [React](https://reactjs.org/), [SolidJS](https://www.solidjs.com/), [Angular](https://angular.io/), [Preact](https://preactjs.com/), [Yew](https://yew.rs/), [Leptos](https://github.com/leptos-rs/leptos) ve [Sycamore](https://sycamore-rs.netlify.app/) için şablonlar içermektedir. Ayrıca [Awesome Tauri repo](https://github.com/tauri-apps/awesome-tauri)'da kendi topluluk şablonlarınızı ve çerçevelerinizi bulabilir veya ekleyebilirsiniz.

Alternatif olarak, mevcut kod tabanınızı hızlı bir şekilde bir Tauri uygulamasına dönüştürmek için [mevcut bir projeye Tauri ekleyebilirsiniz](/tr/guides/create/existing-project).

## Kullanım `create-tauri-app`

`create-tauri-app` komutunu kullanmaya başlamak için projenizi kurmak istediğiniz klasörde aşağıdaki komutlardan birini çalıştırın. Hangi komutu kullanacağınızdan emin değilseniz Linux ve macOS'ta Bash komutunu, Windows'ta ise PowerShell komutunu kullanmanızı öneririz.

import Cta from '../../../_fragments/cta.mdx';

<Cta />

Proje adınızı, ön uç dilinizi, paket yöneticinizi ve ön uç çerçevenizi ve varsa ön uç çerçeve seçeneklerini seçmek için istemleri izleyin.

:::tip[Ne seçeceğinizden emin değil misiniz?]

Başlamak için vanilya şablonu (ön uç çerçevesi olmadan HTML, CSS ve JavaScript) ile başlamanızı öneririz. Daha sonra her zaman [bir ön uç çerçevesi entegre edebilirsiniz](/tr/guides/create/existing-project).

- Ön yüzünüz için hangi dili kullanacağınızı seçin: `TypeScript / JavaScript`
- Paket yöneticinizi seçin: `pnpm`
- UI şablonunuzu seçin: `Vanilla`
- Kullanıcı arayüzünüzü seçin: `TypeScript`

:::

`create-tauri-app` tamamlandıktan sonra projenizin klasörüne gidebilir, bağımlılıkları yükleyebilir ve ardından geliştirme sunucusunu başlatmak için [Tauri CLI](/tr/references/v2/cli) kullanabilirsiniz:

import CommandTabs from '@components/CommandTabs.astro';

<CommandTabs
	npm="cd my-tauri-app
    npm install
    npm run tauri dev"
	yarn="cd my-tauri-app
    yarn install
    yarn tauri dev"
	pnpm="cd my-tauri-app
    pnpm install
    pnpm tauri dev"
	cargo="cd my-tauri-app
    cargo tauri dev"
/>

Şimdi uygulamanızın çalıştığı yeni bir pencerenin açıldığını göreceksiniz.

**Tebrikler!** Tauri uygulamanızı yaptınız! 🚀
