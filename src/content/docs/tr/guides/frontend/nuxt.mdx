https://nuxt.com adresinden Nuxt hakkında daha fazla bilgi edinin. Bu kılavuz Nuxt 3.7 itibariyle doğrudur.

## Kontrol Listesi

- SSG'yi `ssr: false` olarak ayarlayarak kullanın. Tauri sunucu tabanlı çözümleri desteklemez.
- Host'u `0.0.0.0`.
- `tauri.conf.json`da `dist/`ı `distDir` olarak kullanın.
- `nuxi created` kullanarak derleyin.
- (İsteğe bağlı): `nuxt.config.ts` de `telemetry: false` ayarını yaparak telemetriyi devre dışı bırakın.

2. Nuxt yapılandırmasını güncelleyin:

```ts
export default defineNuxtConfig({
	// (optional) Enable the Nuxt devtools
	devtools: { enabled: true },
	// Enable SSG
	ssr: false,
	vite: {
		// Better support for Tauri CLI output
		clearScreen: false,
		// Enable environment variables
		// Additional environment variables can be found at
		// https://tauri.app/2/reference/environment-variables/
		envPrefix: ['VITE_', 'TAURI_'],
		server: {
			// Tauri requires a consistent port
			strictPort: true,
			// Enables the development server to be discoverable by other devices for mobile development
			host: '0.0.0.0',
			hmr: {
				// Use websocket for mobile hot reloading
				protocol: 'ws',
				// Make sure it's available on the network
				host: '0.0.0.0',
				// Use a specific port for hmr
				port: 5183,
			},
		},
	},
});
```
