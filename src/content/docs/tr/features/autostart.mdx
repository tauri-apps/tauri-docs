Uygulamanızı sistem başlangıcında otomatik olarak başlatın.

## Desteklenen Platformlar

- Windows
- Mac (via AppleScript or Launch Agent)
- Linux

## Kurulum

Başlamak için autostart eklentisini yükleyin.

<Tabs>
  <TabItem label="Automatic">

Bağımlılığı eklemek için projenizin paket yöneticisini kullanın:

{' '}

<CommandTabs
	npm="npm run tauri add autostart"
	yarn="yarn run tauri add autostart"
	pnpm="pnpm tauri add autostart"
	cargo="cargo tauri add autostart"
/>

  </TabItem>
  <TabItem label="Manual">

1. Eklentiyi `Cargo.toml` içindeki proje bağımlılıklarına eklemek için `cargo add tauri-plugin-autostart` komutunu çalıştırın.

2. Eklentiyi başlatmak için `lib.rs` dosyasını değiştirin:

```rust title="lib.rs" ins={1, 6}
use tauri_plugin_autostart::MacosLauncher;

#[cfg_attr(mobile, tauri::mobile_entry_point)]
fn run() {
    tauri::Builder::default()
        .plugin(tauri_plugin_autostart::init(MacosLauncher::LaunchAgent, Some(vec!["--flag1", "--flag2"]) /* arbitrary number of args to pass to your app */))
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

3. Tercih ettiğiniz JavaScript paket yöneticisini kullanarak JavaScript Guest bağlarını yükleyebilirsiniz:

## Kullanım

`autostart` eklentisi hem JavaScript hem de Rust dillerinde kullanılabilir.

<Tabs>
  <TabItem label="JavaScript">

```js
import { enable, isEnabled, disable } from '@tauri-apps/plugin-autostart';

// Enable autostart
await enable();
// Check enable state
console.log(`registered for autostart? ${await isEnabled()}`);
// Disable autostart
disable();
```

  </TabItem>
  <TabItem label="Rust">

```rust
use tauri_plugin_autostart::MacosLauncher;
use tauri_plugin_autostart::ManagerExt;

#[cfg_attr(mobile, tauri::mobile_entry_point)]
fn run() {
    tauri::Builder::default()
        .plugin(tauri_plugin_autostart::init(
            MacosLauncher::LaunchAgent,
            Some(vec!["--flag1", "--flag2"]),
        ))
        .setup(|app| {
            // Get the autostart manager
            let autostart_manager = app.autolaunch();
            // Enable autostart
            let _ = autostart_manager.enable();
            // Check enable state
            println!("registered for autostart? {}", autostart_manager.is_enabled().unwrap());
            // Disable autostart
            let _ = autostart_manager.disable();
            Ok(())
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

  </TabItem>
</Tabs>

## İzinler

Varsayılan olarak tüm eklenti komutları engellenir ve erişilemez.
`capabilities` yapılandırmanızda bir izinler listesi tanımlamalısınız.

Daha fazla bilgi için [Erişim Kontrol Listesi](/references/v2/acl) bölümüne bakınız.

```json title="src-tauri/capabilities/main.json"
{
  "permissions": [
    ...,
    "autostart:allow-enable",
    "autostart:allow-disable",
    "autostart:allow-is-enabled"
  ]
}
```

| İzin | Açıklama |
| ---------------------------- | ---------------------------------------------------------------- |
| `autostart:allow-disable`    | Önceden yapılandırılmış bir kapsam olmadan devre dışı bırakma komutunu etkinleştirir.|
| `autostart:deny-disable`     | Önceden yapılandırılmış bir kapsam olmadan disable komutunu reddeder.                |
| `autostart:allow-enable`     | Önceden yapılandırılmış bir kapsam olmadan enable komutunu etkinleştirir.            |
| `autostart:deny-enable`      | Önceden yapılandırılmış bir kapsam olmadan enable komutunu reddeder.                 |
| `autostart:allow-is-enabled` | is_enabled komutunu önceden yapılandırılmış bir kapsam olmadan etkinleştirir.        |
| `autostart:deny-is-enabled`  | Önceden yapılandırılmış bir kapsam olmadan is_enabled komutunu reddeder.             |
