Mobil uygulamanızın QR kodları, EAN-13 ve diğer barkod türlerini taramak için kamerayı kullanmasını sağlar.

## Desteklenen Platformlar

- Android
- iOS

## Kurulum

_Bu eklenti en az **1.64**_ Rust sürümü gerektirir

Başlamak için barkod tarayıcı eklentisini yükleyin.

<Tabs>
  <TabItem label="Automatic">

Bağımlılığı eklemek için projenizin paket yöneticisini kullanın:

{' '}

<CommandTabs
	npm="npm run tauri add barcode-scanner"
	yarn="yarn run tauri add barcode-scanner"
	pnpm="pnpm tauri add barcode-scanner"
	cargo="cargo tauri add barcode-scanner"
/>

  </TabItem>
  <TabItem label="Manual">

1. Eklentiyi `Cargo.toml` içindeki proje bağımlılıklarına eklemek için `cargo add tauri-plugin-barcode-scanner` komutunu çalıştırın.

2. Eklentiyi başlatmak için `lib.rs` dosyasını değiştirin:

```rust title="lib.rs" ins={4}
#[cfg_attr(mobile, tauri::mobile_entry_point)]
fn run() {
    tauri::Builder::default()
        .plugin(tauri_plugin_barcode_scanner::init())
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

3. Tercih ettiğiniz JavaScript paket yöneticisini kullanarak JavaScript Guest bağlarını yükleyin:

  <CommandTabs
      npm="npm install @tauri-apps/plugin-barcode-scanner"
      yarn="yarn add @tauri-apps/plugin-barcode-scanner"
      pnpm="pnpm add @tauri-apps/plugin-barcode-scanner"
  />
  </TabItem>
</Tabs>

## Kullanım

Barkod tarayıcı eklentisi JavaScript'te mevcuttur.

```js
import { scan, Format } from '@tauri-apps/plugin-barcode-scanner';

// `windowed: true` actually sets the webview to transparent
// instead of opening a separate view for the camera
// make sure your user interface is ready to show what is underneath with a transparent element
scan({ windowed: true, formats: [Format.QRCode] });
```

## İzinler

Varsayılan olarak tüm eklenti komutları engellenir ve erişilemez.
`capabilities` yapılandırmanızda bir izinler listesi tanımlamalısınız.

Daha fazla bilgi için [Erişim Kontrol Listesi](/references/v2/acl) bölümüne bakınız.

```json title="src-tauri/capabilities/mobile.json"
{
	"$schema": "./schemas/mobile-schema.json",
	"identifier": "mobile-capability",
	"windows": ["main"],
	"platforms": ["iOS", "android"],
	"permissions": ["barcode-scanner:allow-scan", "barcode-scanner:allow-cancel"]
}
```

| İzin | Açıklama |
| ------------------------------------------- | ------------------------------------------------------------------------------------- |
| `barcode-scanner:allow-cancel`              | Önceden yapılandırılmış bir kapsam olmadan iptal komutunu etkinleştirir.              |
| `barcode-scanner:deny-cancel`               | Önceden yapılandırılmış bir kapsam olmadan iptal komutunu reddeder.                   |
| `barcode-scanner:allow-check-permissions`   | Önceden yapılandırılmış bir kapsam olmadan check_permissions komutunu etkinleştirir.  |
| `barcode-scanner:deny-check-permissions`    | Önceden yapılandırılmış bir kapsam olmadan check_permissions komutunu reddeder.       |
| `barcode-scanner:allow-open-app-settings`   | Önceden yapılandırılmış bir kapsam olmadan open_app_settings komutunu etkinleştirir.  |
| `barcode-scanner:deny-open-app-settings`    | Önceden yapılandırılmış bir kapsam olmadan open_app_settings komutunu reddeder.       |
| `barcode-scanner:allow-request-permissions` | Önceden yapılandırılmış bir kapsam olmadan request_permissions komutunu etkinleştirir.|
| `barcode-scanner:deny-request-permissions`  | Önceden yapılandırılmış bir kapsam olmadan request_permissions komutunu reddeder.     |
| `barcode-scanner:allow-scan`                | Önceden yapılandırılmış bir kapsam olmadan tarama komutunu etkinleştirir.             |
| `barcode-scanner:deny-scan`                 | Önceden yapılandırılmış bir kapsam olmadan tarama komutunu reddeder.                  |
| `barcode-scanner:allow-vibrate`             | Önceden yapılandırılmış bir kapsam olmadan titreşim komutunu etkinleştirir.           |
| `barcode-scanner:deny-vibrate`              | Önceden yapılandırılmış bir kapsam olmadan vibrate komutunu reddeder.                 |
